# AI English Speaking Tutor ⭐⭐⭐⭐⭐

An interactive AI-powered English speaking tutor designed for children to practice pronunciation, improve grammar, and build confidence in speaking English.

## Features

- 🎤 **Speech Recognition**: Uses Web Speech API to capture child's voice input
- ✅ **Grammar Checking**: Real-time grammar analysis with helpful corrections
- 💡 **Sentence Suggestions**: AI-powered suggestions for better sentence structure
- ⭐ **Star Rewards System**: Earn stars for good pronunciation and grammar
- 🔊 **Pronunciation Feedback**: Visual score and audio playback
- 🎨 **Kid-Friendly UI**: Colorful, engaging interface designed for children

## How It Works

1. **Tap to Speak**: Child clicks the microphone button to start recording
2. **Voice Recognition**: The app captures what the child says
3. **AI Analysis**: 
   - Checks grammar and punctuation
   - Calculates pronunciation score
   - Provides corrections and suggestions
4. **Rewards**: Stars are awarded based on performance
5. **Learning**: Child can hear proper pronunciation and see improvements

## Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- Modern browser (Chrome, Edge, or Safari with Web Speech API support)

### Setup

1. Clone or download this project
2. Navigate to the project directory:
```bash
cd AI_English_Speaking_tutor
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

5. Open your browser and navigate to:
```
http://localhost:3000
```

## Usage

1. **Start Speaking**: Click the large microphone button
2. **Speak Clearly**: Say a sentence in English
3. **Get Feedback**: View your pronunciation score and grammar tips
4. **Earn Stars**: Collect stars for good performance
5. **Practice**: Keep practicing to improve your English!
